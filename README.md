## 🎨 Design Philosophy & Aesthetics

This portfolio is crafted under strict **iOS 18 design principles**, utilizing a sleek dark environment with high-contrast, glowing neon accents and premium physical finishes:
- **Glassmorphism:** Frosted translucent panels with rich backdrops, subtle white card borders, and smooth light refractions.
- **Claymorphism:** Deep 3D card textures featuring inner/outer drop shadows (`clay-card-dark`, `clay-card-blue`, etc.) that give components a tactile, physical look.
- **3D Smart Microchip Core:** A central, interactive 3D model designed in React Three Fiber representing a smart processor board. Built with silicon base substrate, gold tracks, connection pins, and a pulsing core that reacts dynamically to mouse-dragging and hover parallax coordinates.
- **Telemetry Circuit Constellations:** A background canvas system featuring floating data nodes that dynamically calculate boundaries and draw line connection grids, reacting to scroll height.
- **Smooth Physics & Springs:** Custom Framer Motion springs for mouse-tracking card glare overlays and animated page transitions.

## ⚙️ Setup & Deployment

### 1. Prerequisites
Ensure you have **Node.js** (v18 or higher) and **npm** installed.

### 2. Local Installation
Clone the repository:
```bash
git clone https://github.com/salarkhan2003/Salar_portfolio.git
cd Salar_portfolio
```

Install dependencies:
```bash
npm install
```

### 3. Development Server
Run locally in development mode:
```bash
npm run dev
```

### 4. Build Pipeline
Generate a compressed, highly optimized production build:
```bash
npm run build
```
